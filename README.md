# smartcomplete

AI-powered code completion for collab-editor and Neovim.

Manual-trigger code generation using grokker backend.

## Features
- Button-triggered completion in collab-editor
- Hotkey-triggered completion in Neovim
- Uses existing grokker infrastructure
- Supports collaborative editing (CRDT-aware)
